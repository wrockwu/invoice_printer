# 介绍
这是一个用于发票打印的简单程序。
程序会自动提取excel某些单元格中的内容填充到发票纸上。可以选择单独打印sheet中某一行的内容，或者连续打印整个sheet中的内容。
目前程序需要excel单元格中的数据按照固定格式来存放。
程序通过windows10和python3.7.3环境验证。其他环境下发现bug，请联系rockwu@126.com。
